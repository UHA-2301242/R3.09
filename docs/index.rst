SAE3.02 - Documentation
=======================

Cette documentation a été réalisé dans le but de proposer une documentation utilisateur et technique de l'architecture de l'application.

Grâce à cette documentation, vous pourrez prendre connaissance des diverses API, méthodes, et autres, mises en place, à la bonne réalisation de ce programme.

La SAE3.02 a pour de mettre en place une architecture client <-> serveur, permettant la communication à travers un socket, permettant l'exécution d'un programme, et le retour de ses logs.
Ainsi, nous devons réaliser :

- Une interface graphique utilisateur
- La mise en place d'une communication stable à l'aide de :py:mod:`sockets`.
- L'exécution de programme par le serveur
- Le retour des logs générés par le programme ayant été lancé sur le serveur, au client
